## Introduction

**`call`**, **`apply`** and **`bind`** are used to invoke functions with a specific `this` context and arguments.

### call
- Invoke the function immediately
- Takes in `comma-separated` arguments

### apply
- Invoke the function immediately
- Takes in `an array of arguments` arguments

<JsEditor initCode={`function add(a, b) {
  return a + b;
}

console.log(add.call(null, 1, 2)); // 3
console.log(add.apply(null, [1, 2])); // 3
`}/>

> An easy way to remember this is C for call and comma-separated and A for apply and an array of arguments.

### bind
- Returns a new function that can be invoked later
- Takes in `comma-separated` arguments

<JsEditor initCode={`function greet(greeting, punctuation) {
    console.log(greeting + ' ' + this.name + punctuation);
}
const person = { name: 'Alice' };
const greetAlice = greet.bind(person);
greetAlice('Hello', '!'); // Output: Hello Alice!
`}/>

## Use Cases

### Context management

<JsEditor initCode={`const withoutCall = {
  name: 'John',
  greet() {
    // this => { name: 'John', greet: fn }
    console.log('Hello, my name is ' + this.name);
  },
};
withoutCall.greet(); // Hello, my name is John
`}/>

Now use `call` / `apply` to override context via `this`

<JsEditor initCode={`const obj = {
  name: 'John',
  greet() {
    console.log('Hello, my name is ' + this.name);
  },
};
const anotherObj = { name: 'Alice' };
obj.greet.call(anotherObj); // Hello, my name is Alice
`}/>

### Function borrowing

Both `call` and `apply` allow borrowing methods from one object and using them in the context of another.

<JsEditor initCode={`function greet() {
    // default: this => window object
  console.log('Hello, my name is ' + this.name);
}

const person1 = { name: 'John' };
const person2 = { name: 'Alice' };

greet.call(person1); // Hello, my name is John
greet.call(person2); // Hello, my name is Alice
`}/>

### Alternative syntax to call methods on objects

`apply` can be used with object methods by passing the object as the first argument followed by the usual parameters.

<JsEditor initCode={`const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

Array.prototype.push.apply(arr1, arr2); // Same as arr1.push(4, 5, 6)
console.log(arr1); // [1, 2, 3, 4, 5, 6]
`}/>
